abstract ServerEvent<T>(Event<T>)

package js.node.net

to Event<T>

import js.node.net.Server

Enumeration of events emitted by the Server objects

Variables

inline read onlyClose:ServerEvent<Void ‑> Void> = "close"

Emitted when the server closes. Note that if connections exist, this event is not emitted until all connections are ended.

inline read onlyConnection:ServerEvent<Socket ‑> Void> = "connection"

Emitted when a new connection is made.

inline read onlyError:ServerEvent<Error ‑> Void> = "error"

Emitted when an error occurs. The 'close' event will be called directly following this event. See example in discussion of server.listen.

inline read onlyListening:ServerEvent<Void ‑> Void> = "listening"

Emitted when the server has been bound after calling Server.listen.
