Enumeration for Writable class events.

Variables

inline read onlyDrain:WritableEvent<Void ‑> Void> = "drain"

If a writable.write(chunk) call returns false, then the drain event will indicate when it is appropriate to begin writing more data to the stream.

inline read onlyError:WritableEvent<Error ‑> Void> = "error"

Emitted if there was an error when writing or piping data.

inline read onlyFinish:WritableEvent<Void ‑> Void> = "finish"

When the end() method has been called, and all data has been flushed to the underlying system, this event is emitted.

inline read onlyPipe:WritableEvent<IReadable ‑> Void> = "pipe"

Lister arguments: src - source stream that is piping to this writable

	This is emitted whenever the `pipe()` method is called on a readable stream,
	adding `this` writable to its set of destinations.

inline read onlyUnpipe:WritableEvent<IReadable ‑> Void> = "unpipe"

Listener arguments: src - source stream that unpiped this writable

	This is emitted whenever the `unpipe()` method is called on a readable stream,
	removing `this` writable from its set of destinations.
